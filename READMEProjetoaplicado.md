Análise de Séries Temporais - Clima

## Sobre o projeto
Este projeto tem como objetivo analisar dados climáticos da cidade de Delhi utilizando técnicas de séries temporais em SQL.

## Técnicas utilizadas
- SQL
- Média móvel
- Análise temporal

## Estrutura
- data → dataset
- sql → consultas
- docs → trabalho acadêmico

## Objetivo
Identificar padrões, tendências e sazonalidade nos dados climáticos.
